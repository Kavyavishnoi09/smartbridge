Project Details

This project classifies rice grain images into multiple rice types using transfer learning. It demonstrates image preprocessing, augmentation, model training, accuracy evaluation, and deployment through a Flask web application.

Objectives

  • Preprocess and augment rice grain images.
  • Apply transfer learning on the rice dataset.
  • Train and evaluate a deep learning model.
  • Save the trained model for future prediction.
  • Build a Flask-based web application for deployment.

Use Cases

Crop Planning

Farmers can identify rice varieties and adapt irrigation and fertilizer plans.

Extension Services

Researchers can classify rice during field visits and trials.

Education

Students and home growers can learn about rice diversity and image AI.
